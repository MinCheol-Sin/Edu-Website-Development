HTML & CSS 기초과정

HTML 코스 요약

HTML (HyperText Markup Language)은 페이지의 콘텐츠와 구조를 정의할 때 사용합니다.

이 과정을 통해 당신은...

  • 열린 태그와 내용, 그리고 닫힌 태그로 이루어진 요소들을 배웁니다.
  • 이미지와 같은 빈요소를 삽입할 수 있습니다.
  • 속성을 통해 특정한 기능이나 동작을 추가할 수 있습니다.
  • 이외에 다양한 요소들이 존재하지만, 이 과정을 통해 HTML의 주요한 개념을 배울 수 있습니다.

HTML의 더 많은 요소들을 알고 싶다면 MDN HTML element reference를 참고하세요.

CSS 코스 요약

CSS (Cascading Style Sheets)는 페이지의 콘텐츠를 꾸밀 때 사용합니다.

이 과정을 통해 당신은...

  • 속성-값의 규칙을 통해 스타일을 정의할 수 있습니다.
  • "style" 속성을 사용하여 스타일을 지정할 수 있습니다.
  • 반복적인 코드를 피하고, 글로벌 표준에 맞는 스타일 코드를 작성할 수 있습니다.
  • 외부의 스타일시트 파일을 연결할 수 있습니다.

CSS의 더 많은 속성과 값들에 대해 알고 싶다면 MDN CSS property reference를 참고하세요.
